Features:

Moderation

TheImperialGod has advanced and superior moderation features
These include:

  1. Kicking users
  2. Banning users
  3. Unbanning users
  4. Purging messages
  5. Adds a role to a user
  6. Removes a role from a user
  7. Locks a channel (for @everyone)
  8. Unlocks a locked channel
  9. Announces a message!
  10. Warns users
  11. Serverinfo - Shows you information about the server
